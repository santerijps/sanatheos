Bible Reader

Search Input

Type in the search bar to find verses. Supported query formats:

  • Book reference — John, Genesis
  • Chapter — John 3, Gen 1
  • Chapter range — Genesis 1-3
  • Verse — John 3:16
  • Verse range — Genesis 1:1-5
  • Multiple verses — Genesis 1:1-3,5,8-10
  • Text search — "grace" (wrap in double quotes)
  • Combined — Romans "faith", Gen 1-3 "light"
  • Multiple terms — separate with ; e.g. John 3:16; Rev 1:1

Typing " auto-inserts a closing quote. Abbreviations like gen, rev, eph are supported.

Browse Books

Click the ☰ menu button (or press Ctrl+I) to open the book index panel. Browse books, chapters, and verses in three columns. Use arrow keys to navigate and Enter to select.

Keyboard Shortcuts

  • Ctrl+K — Focus search input
  • Ctrl+I — Toggle book index panel
  • Escape — Close any open panel
  • ↑ ↓ — Navigate items in the index panel
  • ← → / Tab — Switch columns in the index panel
  • Enter — Select item in the index panel

Translation

This app uses the New King James Version (NKJV) of the Bible, containing all 66 books of the Old and New Testaments.

Data & Storage

Bible text is fetched once from the server and cached locally in your browser using IndexedDB for fast offline access. No data is sent to any third party. Everything runs in your browser.
