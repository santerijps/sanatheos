Search Input

Type in the search bar to find verses. Supported query formats:

  • Book reference — John, Genesis
  • Chapter — John 3, Gen 1
  • Chapter range — Genesis 1-3
  • Verse — John 3:16
  • Verse range — Genesis 1:1-5
  • Multiple verses — Genesis 1:1-3,5,8-10
  • Text search — "grace" (wrap in double quotes)
  • Word boundary — ^grace (starts with), grace$ (ends with), ^grace$ (exact word) — quotes optional
  • Combined — Romans "faith", Gen 1-3 "light"
  • Multiple terms — separate with ; e.g. John 3:16; Rev 1:1

Typing " auto-inserts a closing quote. Abbreviations like gen, rev, eph are supported.

Browse Books

Click the search button (or press Ctrl+I) to open the book index panel. Browse books, chapters, and verses in three columns. Use arrow keys to navigate and Enter to select.

Keyboard Shortcuts

  • Ctrl+K — Focus search input
  • Ctrl+I — Toggle book index panel
  • Escape — Close any open panel
  • ↑ ↓ — Navigate items in the index panel
  • ← → / Tab — Switch columns in the index panel
  • Enter — Select item in the index panel

Settings

Click the menu button to open settings. You can switch between Bible translations (e.g. NHEB, KR38), choose a parallel translation (side-by-side view), set the theme (Light, Dark, or System), and change the application language. All selections are saved across sessions.

Features

  • Highlights — Right-click a verse number and choose a highlight color (yellow, green, blue, pink, orange). Choose “Remove highlight” to clear it.
  • Bookmarks — Click the bookmark icon on a section heading to bookmark the current passage, or right-click a verse number to bookmark that specific verse. Bookmarks are saved across sessions and listed in the Bookmarks tab of the side panel.
  • Copy — Click the 📋 button on a section heading to copy displayed verses. In parallel mode, both translations are included.
  • Swipe navigation — On touch devices, swipe left or right to move between chapters.
  • Print — Use Ctrl+P for a clean, print-optimized layout.
  • Install as app — Sanatheos is a Progressive Web App. Use your browser’s “Install” or “Add to Home Screen” option to install it for quick, offline access.

Data & Storage

Bible text is fetched once from the server and cached locally in your browser using IndexedDB for fast offline access. No data is sent to any third party. Everything runs in your browser.

All available Bible translations are in the public domain.
